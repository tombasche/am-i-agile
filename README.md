## Usage

`npm install`

and ... 

`yarn start`

It should be that simple.


To change the groups to pick from just add a `groups.json` file in the `src/` directory. The structure of this file:

```
{
  "groups": [
    {
      "name": "group 1",
      "members": ["tom", "bob", "bill"]
    }
  ]
}
```
